## Overview

The Centralized API Request Repository provides a unified, always-updated view of every API request across your projects and workspaces.\
Teams can easily explore methods, URLs, associated collections, and vulnerability counts — enabling deeper understanding of API behavior and identifying high-risk endpoints quickly.

## Key Capabilities

<Columns cols={2}>
  <Card title="Unified Request Inventory" icon="list">
    View all API requests from every workspace, project, and imported collection in one centralized repository.
  </Card>
  <Card title="Method & URL Visibility" icon="file-lines">
    Inspect HTTP methods, endpoint URLs, content types, and request structures for complete clarity.
  </Card>
  <Card title="Collection Mapping" icon="folder">

  <Card title="Vulnerability Counts" icon="shield-halved">

  <Card title="Cross-Project Visibility" icon="layer-group">

  </Card>

## Value

Gain full visibility into all API interactions across your ecosystem.\
The centralized repository helps teams identify high-impact endpoints, understand usage patterns, and streamline API vulnerability analysis.
