## Overview

The Customizable YAML Rule Engine empowers teams to define their own API security testing logic using a flexible YAML-based syntax.\
From request transforms and behavioral validations to CWE-driven response analysis, the engine enables deep, customizable, and highly targeted API vulnerability detection for any endpoint or workflow.
