The **Scanners** page lists all registered vulnerability scanners and allows you to create new ones.  
Each scanner displays its current status, URL, and vulnerability counts discovered by that scanner.

<Frame caption="Scanner List Overview">
  <img src="/images/vs/products/list.png" />
</Frame>

---

## Overview

The Scanners page shows:

- **Scanner name**  
- **Creation timestamp**  
- **Status badge** (e.g., *Active*)  
- **Scanner URL**  
- **Vulnerability counts** (C0 · H0 · M0 · L0)

You can also **search scanners** using the Search bar at the top.

---

## Scanner Card Breakdown

Each scanner card displays:

- **Name** (e.g., *Default Scanner*)  
- **Created On** timestamp  
- **Status Indicator** (*Active*)  
- **Scanner URL** (e.g., `http://vscanner:3018`)  
- **Vulnerability Summary**  
  - C0 = Critical  
  - H0 = High  
  - M0 = Medium  
  - L0 = Low  
- **Delete Scanner** action (trash icon)

---

## Create Scanner

Users can add a new scanner using the **Create Scanner** button.

<Frame caption="Create Scanner Modal">
  <img src="/images/vs/products/create.png" />
</Frame>

### Fields in the Create Scanner Form

- **Scanner Name**  
  A user-defined name for the scanner.

- **Scanner URL**  
  The live and externally accessible URL of the scanner.

### Steps to Create a Scanner

1. Click **Create Scanner**.  
2. Enter a **Scanner Name**.  
3. Enter the **Scanner URL**.  
4. Click **Create Scanner** to register the scanner.
