## Overview

The **IP Addresses** section in AIM provides a complete inventory of all IPs discovered across your organization.  
It helps teams understand **network exposure, geolocation, vulnerabilities, and ownership**, enabling faster risk identification and response.

## Dashboard View

## Key Insights

### External Reachability
Shows how many IPs are:
- Reachable (externally accessible)  
- Unreachable  

This helps identify publicly exposed infrastructure.

---

### Location Distribution
Breakdown of IPs by geography:
- Country  
- Region  
- Global distribution  

Useful for understanding infrastructure spread and unexpected locations.

---

### Vulnerable vs Safe
Quick view of:
- Vulnerable IPs  
- Safe IPs  

Helps prioritize remediation efforts.

---

### Vulnerability Severity Distribution
Displays severity levels:
- High  
- Medium  
- (and others if present)

---

### Network Scope
Classifies IPs as:
- External  
- Internal  

---

### ASN Distribution
Shows which providers host your infrastructure:
- AWS  
- Google Cloud  
- Hosting providers  

Helps identify third-party exposure risks.

## Inventory Table

The inventory table is the **primary working view** for IP management.

### Key Columns

| Column | Description |
|--------|-------------|
| **IP Address** | The discovered IP |
| **Location** | Country and city of the IP |
| **Reachability** | Whether the IP is externally reachable |
| **ISP / ASN** | Hosting provider or network |
| **Vulnerabilities** | Number of associated vulnerabilities |
| **Owner** | Assigned owner |
| **Business Unit** | Associated business group |
| **Associations** | Linked assets (domains, services, etc.) |
| **Detected On** | First and last detection timestamps |

---

## Search and Filters

Quickly refine IP visibility using:

- **Is Vulnerable** — Show IPs with vulnerabilities  
- **IP Version** — IPv4 / IPv6 filtering  
- **Locations** — Filter by geography  
- **ISPs** — Filter by hosting provider  
- **Search** — Locate specific IPs  

These filters help isolate high-risk or unusual infrastructure quickly.

---

## Why This Matters

The IP module gives you direct visibility into your **network attack surface**:

- Identify **externally exposed infrastructure**  
- Detect **unexpected geolocations or hosting providers**  
- Track **IP-level vulnerabilities and risks**  
- Understand **asset relationships across IPs**  
- Enforce **ownership and accountability**
