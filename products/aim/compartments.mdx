## Compartment Structure & Status

Each compartment entry includes:

- **Compartment Name** â€“ such as *Systems-Production*, *OCMDevelopment*, or *Platform-Staging*  
- **Lifecycle Status** â€“ Active / Deleted, based on the OCI sync  
- **ARN Identifier** â€“ full OCI resource ID for traceability  
- **Associated Assets** â€“ number of assets mapped inside the compartment

<Info>
Active compartments represent live cloud environments. Deleted compartments indicate resources removed from OCI but retained in AIM for audit history.
</Info>
