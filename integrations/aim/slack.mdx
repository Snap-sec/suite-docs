## 1. Prerequisites

Before integrating Slack with Snapsec, ensure you have:

- A Slack workspace  
- Permission to create a Slack App or Incoming Webhook  
- A Snapsec account with admin access  
- The Slack **Webhook URL** for the channel you want notifications sent to

## 2. Create a Slack Incoming Webhook

Follow these steps to create the webhook Slack will use to receive alerts:

1. Visit: https://api.slack.com/messaging/webhooks

3. Name the app (example: `Snapsec Alerts`)

6. Click **Add New Webhook**  
7. Select a channel (example: `#security-alerts`)  
8. Copy the generated **Webhook URL**

## 4. Test the Slack Integration

In Snapsec:

1. Click **Test Connection**  
2. Snapsec sends a sample alert to your Slack channel  
3. You should see a message like:

> **Snapsec Alert Test**  
> Your Slack integration is working correctly.

If nothing appears:

- Recheck the webhook URL  
- Ensure the Slack channel permissions allow bot messages  
- Validate internet restrictions or firewall rules
